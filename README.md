# SmartMed Analytics

AI-Powered Healthcare Risk Prediction & Analytics Platform

---

## 📌 Project Overview

SmartMed Analytics is an intelligent healthcare web application that predicts the risk of major diseases using Machine Learning models. The system helps users monitor their health through AI-powered analytics, healthcare dashboards, recommendations, and prediction reports.

The platform currently supports:

* ❤️ Heart Disease Prediction
* 🩸 Diabetes Prediction
* 🩺 Kidney Disease Prediction

Built using:

* React.js frontend
* Flask backend
* Machine Learning with Scikit-learn
* MongoDB database

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Vaishnavidasyam/healthcare-risk-prediction.git
```

---

# 2️⃣ Backend Setup

## Navigate to backend

```bash
cd backend
```

## Install dependencies

```bash
pip install -r requirements.txt
```

## Run Flask server

```bash
python app.py
```

Backend runs on:

```bash
http://127.0.0.1:5000
```

---

# 3️⃣ Frontend Setup

## Open new terminal

```bash
cd frontend
```

## Install packages

```bash
npm install
```

## Start React app

```bash
npm start
```

Frontend runs on:

```bash
http://localhost:3000
```

---

# 🤖 Train ML Models

Run model training scripts:

```bash
python ml/train_heart.py
python ml/train_diabetes.py
python ml/train_kidney.py
```

Generated models:

```bash
models/
├── heart_pipeline.pkl
├── diabetes_pipeline.pkl
└── kidney_pipeline.pkl
```
